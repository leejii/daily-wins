# 입력 폼과 구글 시트를 연결해 사용자의 일일 목적, 루틴 등을 기록하는 사이트.

[목적]
이 프로젝트는 사용자가 매일 수행한 습관(운동, 스트레칭, 영어공부, 책읽기 등)을 간단한 폼 입력을 통해 기록하고, 해당 데이터가 자동으로 구글 시트에 저장되도록 한다.
목표는 사용자가 최소한의 입력으로 하루의 성취를 기록하고, 꾸준함을 시각적으로 확인할 수 있도록 하는 것이다.

[전체구조]
사용자 - 웹페이지에서 폼 작성 - Google Apps Script → Google Sheets에 저장

[구현스택]
- HTML, CSS, JavaScript
- 데이터베이스는 google Sheets를 Apps Script로 연결해서 사용

[요구사항]
- Form은 날짜 | 수행 목표 | 수행 시간 | 메모
- 날짜는 오늘 날짜가 default. 날짜 변경 시 캘린더에서 날짜 선택 가능
- 수행 목표는 운동, 스트레칭, 독서, 영어, 말, 기타 -> 이 항목으로 선택박스가 주어지고 기타 선택 시 사용자가 직접 입력할 수 있는 input이 생긴다.
- 수행 시간은 최소 10분부터 5분 단위이며 최대 1시간까지 선택
- 메모는 사용자가 간단한 메모를 입력할 수 있다.
- 저장고 취소 버튼이 있고 취소 버튼 입력 시 입력한 값 초기화
